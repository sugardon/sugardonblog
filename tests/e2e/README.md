# `tests/e2e`

## Run

### development

1. Run app
   ```bash
   yarn dev
   ```
1. Run test
   ```bash
   yarn test:e2e
   ```

Also, you can use `blog.sugardon.com`

1. Set url
   ```bash
   export PLAYWRIGHT_BASE_URL="https://blog.sugardon.com"
   ```
1. Run test
   ```bash
   yarn test:e2e
   ```
